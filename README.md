# Puedes usar este proyecto como base para tu introducción al mundo del desarrollo web.

## Como correr el Proyecto.
1. Puedes clonar el repositorio, o descargar el código en archivo zip.
2. Descomprime el archivo zip.
3. Abrir tu editor de código.
4. Ingresar a la carpeta.
5. Cambiar la información por la tuya.
